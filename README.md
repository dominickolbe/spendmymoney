## Install dependencies
make sure you have installed all dependencies before moving on
```
$ yarn
```

## Development
develop application
```
$ yarn start
```

## Production
build application
```
$ yarn build
```

## Deployment
this command first builds the application and then deploys it to firebase
```
$ yarn deploy
```
